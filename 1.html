html이란 무엇인가?

Hypertext Markup Language (html) is the standard language for creating webpages and web applications. Web browsers recive HTML documents from a web server or from local storage and render them into multimedia web pages.

HTML describes the struvture of a web page semantically and originally included cues for the appearance of the doucment.
